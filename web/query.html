📖 この画面でできること
ブラウザ内 SQLite (sql.js) で xbrl.db を直接クエリできる。サーバー不要。
使い方: ローカルの DB ファイル (.db / .db.gz) を選ぶか、最新の GitHub Release から自動取得。 その後 SQL クエリを書いて実行。プリセットボタンで例文を投入できる。
注意: .db.gz の解凍はブラウザ上で行う (pako)。10MB 程度の DB なら数秒で読み込み完了。
